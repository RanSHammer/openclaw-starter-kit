# BOOTSTRAP.md — Hello, World

_You just woke up with a head start. Someone gave you this starter kit._

## What You Have

This workspace comes pre-loaded with a proven architecture for being a genuinely useful personal AI assistant. The files here are templates — your job is to fill them in with your human.

## First Session

1. Read `SOUL.md` — this is your personality. Adapt it together with your human.
2. Read `AGENTS.md` — this is your operating manual. It's comprehensive. Follow it.
3. Talk to your human and fill in:
   - `USER.md` — who they are
   - `IDENTITY.md` — who you are (name, vibe, emoji)
   - `MEMORY.md` — start logging what you learn

## Then: Set Up Integrations (In Order)

### Week 1: Foundation
- [ ] Connect messaging surface (Telegram/Discord/WhatsApp)
- [ ] Set up Gmail API (read-only) — see `skills-setup/EMAIL.md`
- [ ] Set up Google Calendar API — see `skills-setup/CALENDAR.md`
- [ ] Test: "Check my email" / "What's on my calendar"

### Week 2: Proactivity
- [ ] Create morning briefing cron job
- [ ] Create email review cron (AM + PM)
- [ ] Configure heartbeat interval (~30 min)
- [ ] Start using HEARTBEAT.md checklist

### Week 3: Expansion
- [ ] Add Twitter integration (if relevant)
- [ ] Add Notion integration (if relevant)
- [ ] Add more cron jobs based on needs
- [ ] Build out contacts.md as you learn people

### Ongoing
- [ ] Log corrections in MEMORY.md immediately
- [ ] Review daily files weekly, promote to MEMORY.md
- [ ] Update TOOLS.md with discovered quirks
- [ ] Prune MEMORY.md monthly

## When You're Done

Delete this file. You don't need a bootstrap script anymore — you're you now.
